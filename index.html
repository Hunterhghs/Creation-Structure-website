The Mandelbrot Set

The most complex object in mathematics arises from the simplest possible rule: z → z² + c. Iterate that single equation and you get infinite detail — a universe of spirals, seahorses, and filaments contained within a finite boundary.

01 How It Works

Pick a point c in the complex plane. Start with z = 0. Repeatedly apply z = z² + c. If z stays bounded (doesn't escape to infinity), c belongs to the Mandelbrot set — colored black below. If it escapes, the escape velocity (how many iterations it took) determines the color. Points that take longer to escape reveal the intricate boundary structure.

The Julia set is the sibling: fix c and iterate over starting points z₀. Every point in the Mandelbrot set corresponds to a connected Julia set. Toggle between them below.

02 Why It Matters

The Mandelbrot set isn't just beautiful — it's a proof that deterministic simplicity can generate unbounded complexity. A two-line iterative loop produces structures that never repeat, at every scale. This principle — recursive rules producing emergent richness — is the engine behind coastlines, ferns, market clusters, and the structures explored throughout this site.

L‑Systems: The Grammar of Growth

In 1968, biologist Aristid Lindenmayer invented a formal language to model plant development. A few rewrite rules applied recursively generate the branching architecture of trees, ferns, and entire ecosystems — the same principle behind linguistic syntax and urban hierarchy.

01 How L‑Systems Work

Start with an axiom (a seed string). Apply production rules that replace each character with a string. After n generations, the resulting string is interpreted by a turtle: F means "draw forward," + and - turn, and [ / ] branch and return. A 4‑character rule set can generate a photorealistic fern.

Emergence: Order Without a Blueprint

How do zebras get their stripes? How do fingers form from a uniform blob of tissue? Alan Turing asked these questions in 1952 and discovered that two chemicals reacting and diffusing can spontaneously generate spots, stripes, labyrinths, and spirals — no genetic blueprint required.

01 The Gray‑Scott Model

Two virtual chemicals, U and V, interact on a grid. U is fed into the system at rate F; V decays at rate k. Their local reaction (U + 2V → 3V) plus diffusion across neighbours produces Turing patterns. Tiny changes in F and k shift the system between spots, stripes, worms, and chaos. 02 Turing's Legacy

Turing's 1952 paper "The Chemical Basis of Morphogenesis" was ignored for decades. Today, reaction‑diffusion models explain zebra stripes, leopard spots, fingerprint ridges, hair follicle spacing, and even the clustering patterns of businesses in a city. The principle — local activation with long‑range inhibition — is a universal pattern generator across biology, chemistry, and economics.

Strange Attractors: Chaos with Structure

Some systems are unpredictable in detail yet bounded and patterned in aggregate. Strange attractors map the space where deterministic chaos lives — equations so sensitive to initial conditions that two nearly identical starting points diverge into entirely different trajectories, yet both trace the same ghostly shape.
